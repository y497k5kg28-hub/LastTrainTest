# RPG Game as a Telegram Mini App

## Setup Instructions

1. **Clone the Repository**  
   Clone this repo to your local machine using:
   ```bash
   git clone https://github.com/y497k5kg28-hub/LastTrainTest.git
   ```

2. **Install Dependencies**  
   Navigate to the project directory and install the required dependencies:
   ```bash
   cd LastTrainTest
   npm install
   ```

3. **Set Up Telegram Bot**  
   - Create a new bot on Telegram by searching for the "BotFather".
   - Follow the instructions provided by BotFather to get your bot token.

4. **Configure Environment Variables**  
   Create a `.env` file in the root directory of the project and add your bot token:
   ```plaintext
   TELEGRAM_BOT_TOKEN=your_bot_token_here
   ```

5. **Run the Application**  
   Start the application using:
   ```bash
   npm start
   ```

6. **Test the App**  
   Open Telegram and start a chat with your bot. Follow the commands and prompts to play the RPG game.

## Important Notes
- Ensure your local environment meets the requirements specified in the project documentation.
- For further assistance, refer to the project's Wiki or contact the project maintainers.
